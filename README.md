# Digital Clock

A simple digital clock web application built using HTML, CSS, and JavaScript.  
It shows the current time and updates every second automatically.

## Features
- Displays current hours, minutes, and seconds
- Updates time every second
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
- JavaScript Date object is used to get the current time
- setInterval() updates the time every second
- DOM manipulation is used to display time on the screen
